# Benchmark Seeds

This folder keeps small, checked-in benchmark metadata used to seed dev-only
agent-loop replay generation.

- `agent-loop-benchmark-cases.v1.json` is a curated starter pack, not a full
  benchmark mirror.
- Current focus is `Terminal-Bench` starter tasks because they are the closest
  public match to OPApp's local terminal-first coding loop.
- `SWE-bench` stays deferred until the local loop is stable enough to benefit
  from heavier repo-level issue cases.
