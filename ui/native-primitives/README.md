# @opapp/ui-native-primitives

Shared native UI tokens and primitives for capability surfaces.

## Design Baseline

- [Design Spec v1](./docs/design-spec-v1.md)
- [Component System v1](./docs/component-system-v1.md)

## Exported Contracts

- Tokens: `appPalette`, `appRadius`, `appSpacing`, `appTypography`
- Tone tokens: `appTonePalette`
- Layout: `appLayout`
- State set: `appInteractionStates`
- Component constraints: `appComponentContractsV1`
- Tactical surface semantic palette: `tacticalSurfacePaletteV1`
- Primitives: `AppFrame`, `SectionCard`, `ChoiceChip`, `ActionButton`, `StatusBadge`, `SignalPill`, `TimelineStep`, `FilterChip`, `InfoPanel`
